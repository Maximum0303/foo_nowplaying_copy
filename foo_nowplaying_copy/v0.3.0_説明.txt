NowPlaying Copy & Artwork v0.3.0

追加機能：
・MP3、FLAC、M4Aなどの音楽ファイルに埋め込まれたフロントカバーを抽出
・ZIP内のMP3に埋め込まれた画像も、foobar2000のアートワークAPI経由で取得を試行
・画像形式をJPEG / PNG / GIF / WebP / BMP / TIFFのヘッダーから判別
・完了画面に「埋め込み」「同じフォルダ」「ZIP内」の取得元を表示
・埋め込み画像がなければ、従来のフォルダ画像またはZIP内画像へ自動フォールバック

取得優先順位：
1. 音楽ファイル内の埋め込みフロントカバー
2. 通常フォルダ内の cover / folder / front
3. ZIP内の cover / folder / front

ビルド：Release / x64 で「ソリューションのリビルド」を実行してください。
